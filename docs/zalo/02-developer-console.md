# Zalo Developer Console — Núi Bà Đen

## Mục tiêu
- Ghi lại các mục cần kiểm tra trong Zalo developer console.
- Làm checklist cấu hình cho app hiện có.

## Thực tế hiện tại
- Đã có account vào app console.
- Mini app đã tồn tại ở chế độ DEV.

## Cần kiểm tra trong console
- App ID
- Tên app
- Icon / branding
- OA liên kết nếu có
- Domain / environment settings
- Build/publish history
- Permission settings nếu có

## Liên quan đến dự án này
- Xác minh app shell đang dùng đúng cho Núi Bà Đen.
- Xác minh branding đúng tên/logo.
- Xác minh cấu hình phục vụ 2 ngôn ngữ nếu có chỗ liên quan.
- Xác minh môi trường DEV/STAGING/PROD nếu console hỗ trợ.

## Dữ liệu nên ghi lại nội bộ
- App ID
- Mốc tạo app
- Owner/quyền truy cập
- Liên kết OA
- Build hiện tại
- Link console chính

## Không lưu vào docs public
- Secret
- Token
- Credentials
- Cookie/session data

## Checklist thao tác
- Mở app console
- Kiểm tra thông tin app
- Kiểm tra lịch sử hoạt động/build nếu có
- Kiểm tra các mục permission/review
- Kiểm tra phần Checkout SDK chỉ để xác nhận không dùng

## Câu hỏi mở
- Ai là owner chính của app?
- Có cần phân quyền thêm cho team vận hành không?
